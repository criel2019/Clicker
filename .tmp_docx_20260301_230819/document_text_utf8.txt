플랫폼

한 줄 요약
카드 한 장짜리 소설 같은 클리커. 텍스트와 일러스트만으로 감정을 판다.
1.1 핵심 컨셉
기억을 잃고 어린이 모습으로 난세의 중세를 살아가는 전설 속 신수들. 플레이어는 이세계 환생자로서 신수들을 하나씩 구출하고 훈련시키며 정들게 된다. 그러나 신수가 충분히 강해지면 플레이어는 이유도 모른 채 다음 지역으로 갑자기 순간이동된다. 반복되는 이별과 재회. 그리고 감춰진 진실.
1.2 핵심 감정선
플레이어: 자각 없는 무의식적 가해자 — 신수와 정들고, 아무것도 모른 채 떠난다
왜소한 신: 모든 진실을 알지만 말을 못한 채 옆에서 지켜본다
신수들: 주인 없이 남겨진 후 스스로 여행을 떠나 플레이어를 다시 찾아온다
엔딩: 반복된 이별 끝에 진실이 드러나는 카타르시스
1.3 수익화 개요
스킨 판매 — 캐릭터당 5개 내외 (직접 구매)
호감도 아이템 판매
훈련도구 최상급 기간제 판매
스킨 가챠권 — 황금 고블린 희귀 드랍
2. 세계관
2.1 세계 설정
중세 난세. 전쟁과 혼란이 끊이지 않는 세계. 전설 속 신수들이 존재한다는 이야기가 전해지지만 아무도 실제로 본 적 없다. 신수들은 기억을 잃은 채 어린이 모습으로 세상 속에 흩어져 살고 있다.
2.2 황룡: 두 면을 가진 하나의 존재
📌 진짜 황룡과 가짜 황룡은 같은 존재의 두 면이다. 시스템적 면(가짜)과 감정적 면(진짜)으로 분리되어 있다.
진짜 황룡 — 왜소한 신
전임자(플레이어의 전생)가 키웠던 신수 중 하나로, 전임자가 떠난 후 신격화된 존재. 전임자를 다시 보고 싶은 그리움에 자신의 신적인 힘을 써서 전임자를 현세에 전생시켰다. 그 힘을 쓴 대가로 왜소하고 말을 못하는 모습이 되었다. 항상 플레이어 곁에 말없이 따라다닌다.
외형: 작고 왜소함. 귀여운 생명체 같은 모습
특징: 말을 못한다. 모든 진실을 알지만 전달할 수 없다
감정: 플레이어를 그리워하여 불렀지만, 매번 이별을 반복하는 것을 옆에서 지켜봐야 한다
순간이동 시: 플레이어와 함께 이동한다. 신수들 눈에는 둘이 함께 떠나는 것으로 보인다
가짜 황룡 — 시스템적 신
진짜 황룡이 자신의 시스템적 면을 분리해 만든 인형. 4신수 각성 완료 전까지는 시스템 창으로만 존재하다가, 이후 어린이 모습으로 실체화된다.
성격: 공허하다. 자신이 왜 존재하는지 모른다
역할: 플레이어에게 신수 훈련 미션을 부여하고, 훈련 방법을 지시한다
진실 인지: 진짜 황룡의 존재를 모르지만 스토리 진행하며 어렴풋이 느끼게 된다
결말: 진짜 황룡과 합쳐져 완전한 황룡이 된다. 두 자아 모두 유지되며 시프팅 가능
2.3 플레이어
전임자의 환생. 전생의 기억은 전혀 없다. 어느 날 이 세계에 눈을 떠보니 시스템 창이 떠 있고, 옆에 말 못하는 이상한 존재(왜소한 신)가 붙어있다. 가짜 황룡의 지시에 따라 신수들을 키우기 시작하지만 왜 그래야 하는지 이유를 모른다. 훈련 방법은 가짜 황룡이 지시해주는 것을 통해 머릿속에 지식으로 들어온 것처럼 처리된다.
2.4 신수들
전설 속 존재들이지만 기억을 잃고 어린이 모습으로 세상에 흩어져 있다. 플레이어와도, 왜소한 신과도 초면. 자신이 신수인 줄도 모른 채 각자의 처지에서 살아간다. 플레이어에게 거두어지면서 훈련을 통해 성장하고, 성장할수록 플레이어에게 마음을 열기 시작한다.
2.5 순간이동: 반복되는 비극
신수가 충분히 강해지면(5성 + 각성 완료) 플레이어는 이유도 모른 채 왜소한 신과 함께 다음 지역으로 갑자기 순간이동된다. 신수는 그 자리에 남겨진다. 작별 인사도 없이.
플레이어: 이유를 모른다
왜소한 신: 알지만 말을 못한다. 사실 이 순간이동은 진짜 황룡이 만든 시스템의 결과물이지만, 감정적으로는 막고 싶어한다
신수들: 어제까지 같이 있었는데 오늘 아침에 일어나니 둘 다 사라져있다. 왜소한 신이 플레이어와 함께 떠났다는 것에 배신감을 느낄 수 있다
이후: 남겨진 신수들은 각자의 여행을 통해 플레이어를 다시 찾아온다
2.6 전체 스토리 흐름
시작
시스템 창만 뜸 (가짜 황룡 — 신수를 키워라, 미션 부여)
왜소한 신이 말없이 옆에 붙어있음. 정체 불명
4신수 중 어느 순서로든 선택해서 구출 시작
4신수 반복 구조 (각자 독립 스토리)
구출 → 훈련 → 성장(레벨/성급) → 각성 아이템 획득 → 각성 → 순간이동
순간이동 시 해당 신수는 남겨짐, 플레이어와 왜소한 신은 새 지역으로 이동
4신수 각성 완료 후
가짜 황룡이 어린이 모습으로 실체화 등장 (5번째 신수)
동일한 구조로 키우기 시작
진실이 하나씩 밝혀지기 시작
왜소한 신의 정체가 드러남
가짜 황룡과 왜소한 신이 합쳐짐
엔딩: 완전한 황룡이 처음으로 말을 건넨다 — "쭉 이날만을 기다렸어요"
엔딩 이후
일상물 에피소드 시작
두고 온 신수들이 하나씩 찾아오며 관계 이벤트 전개
3. 캐릭터
📌 모든 신수는 어린이 모습이다. 각성 전후 외형 동일. 각성 시 이펙트만 변화 (포켓몬식).
3.1 청룡
성격
호쾌한 여장부. 허당끼가 있음. 본인은 엄청난 존재라 자부하지만 현실은 허드렛일
처음 처지
해적단 말단. 큰소리치지만 아무도 인정 안 해줌
구출 방식
플레이어가 해적단을 부수고 함께 탈출
각성 이후
진짜 여장부. 위압감 있는 실력자로 거듭남
탑 스토리
호쾌한 성격의 반대 — 답답하고 소심한 적들과의 전투. 자신의 성격 극복 서사
3.2 백호
성격
도도함. 자신이 누구보다 고귀하다고 생각함. 굽히지 않음
처음 처지
귀족 집안 하녀. 도도하게 굴다가 혼나도 꺾이지 않음
구출 방식
플레이어가 같은 하녀로 잠입해 함께 탈출
각성 이후
전우치 스타일. 매우 빠르고 묵직한 공격 + 도술
탑 스토리
도도한 성격의 반대 — 자기보다 잘난 척하는 적들과의 전투. 자존심 극복 서사
3.3 주작
성격
귀엽고 맘씨 좋음. 남을 잘 챙기고 이용당하기 쉬운 성격
처음 처지
고아원 왕따. 모두를 챙겨주지만 오히려 이용당하고 따돌림 받음
구출 방식
플레이어가 고아원에서 입양
각성 이후
성격은 그대로지만 플레이어에 대한 애정이 무거워짐. 집착에 가까운 애정
탑 스토리
맘씨 좋은 성격의 반대 — 약자를 괴롭히는 적들과의 전투. 분노와 보호 본능 극복 서사
3.4 현무
성격
바보같음. 아무 생각 없이 평화롭게 삶
처음 처지
호수 위에 수달처럼 둥둥 떠서 살고 있음
구출 방식
플레이어가 먹이로 유인해서 데려옴
각성 이후
독뱀을 다루고 땅 구르기 한 번에 온 산이 울리는 괴력가. 성격은 여전히 바보같음
탑 스토리
바보같은 성격의 반대 — 머리 쓰는 트릭 계열 적들과의 전투. 꾀에 당하지 않는 서사
3.5 가짜 황룡 (5번째 신수)
성격
공허함. 자신이 왜 존재하는지 모름
외형
어린이 모습으로 실체화
등장 시점
4신수 각성 완료 후
역할
4신수를 키우는 동안 시스템 창으로만 존재했던 가짜 황룡이 실체화된 것
진실
진짜 황룡(왜소한 신)이 만든 인형. 진짜와 어렴풋이 연결되어 있음
결말
왜소한 신과 합쳐져 완전한 황룡이 됨. 두 자아 모두 살아있으며 시프팅 가능
3.6 왜소한 신 (진짜 황룡)
외형
작고 왜소한 생명체. 귀여운 모습
특징
말을 하지 못한다
정체
전임자(플레이어 전생)가 키웠던 신수. 신격화된 황룡
동기
전임자를 다시 보고 싶은 그리움. 신적인 힘을 써서 전임자를 전생시킨 대가로 왜소해짐
행동
항상 플레이어 옆에 있음. 순간이동 시에도 함께
고독
모든 진실을 알지만 말을 못한다. 플레이어가 신수들과 정들고 떠나는 것을 매번 옆에서 지켜봐야 함
4. 시스템 설계
4.1 성장 시스템
레벨 & 성급 구조
레벨 1~50 → 1성
레벨 51~100 → 2성
레벨 101~150 → 3성
레벨 151~200 → 4성
레벨 201~250 → 5성
5성 달성 + 각성 아이템 보유 시 각성 가능
경험치 획득
탭 훈련: 탭 1회당 경험치 1 획득
방치 수익: 시간이 지나면 자동으로 경험치 적립. 탭보다 느린 속도
성장 보상: 레벨업 시 경험치 배율 아이템 지급 → 후반 레벨업 속도 보정
각성
각성 아이템: 스토리 진행 중 각성 이벤트 직전 스테이지에서 획득
조건: 5성 달성 + 각성 아이템 보유
외형 변화 없음 — 어린이 모습 그대로 유지 (포켓몬식)
이펙트 변화만 있음 — 각성 후 전투/훈련 이펙트가 달라짐
각성 완료 = 순간이동 트리거
4.2 전투 시스템
📌 다이스는 내부적으로만 굴러간다. 유저에게 수치가 보이지 않는다. 텍스트 결과만 출력된다.
기본 구조
스토리 텍스트 진행 → 적 등장 시 탭 버튼 출현 → 탭 1회 = 내부 다이스 1회 + 텍스트 결과 출력
내부 다이스 구조 (유저에게 비공개)
2d6 (6면체 주사위 2개) 자동 롤
결과 2~12 중 해당 구간의 텍스트 묘사만 화면에 출력
2~4: 실패 묘사 (빗나감, 역효과 등)
5~7: 평타 묘사
8~10: 좋은 공격 묘사
11~12: 크리티컬 묘사
성급 보정 (다이스 결과에 내부적으로 추가)
1성: +0
2성: +1
3성: +2
4성: +3
5성: +4
각성: +6
탑 능력치 보정 (탑 클리어 보상으로 획득, 별도 적용)
공격 보정: 다이스 결과에 추가
회피 보정: 적 공격을 빗나가게 할 확률
방어 보정: 적 공격 데미지 감소
적 시스템
적마다 HP 보유
HP 상태는 텍스트로만 표현 (수치 미표시)
멀쩡함 / 약간 지쳐보임 / 많이 지쳐보임 / 거의 쓰러질 것 같음 / 쓰러짐
적마다 공격 빈도 다름 — 약한 적 / 중간 적 / 보스
적은 일정 빈도로 자동 공격 다이스를 내부 롤 → 텍스트로만 결과 출력
보스는 스토리 흐름에 맞게 중간중간 배치 (특정 챕터 끝이 아닌 자연스러운 배치)
캐릭터 HP 표현 (이미지)
📌 캐릭터는 어린이 모습이므로 표정 변화로 HP를 표현한다. HP 수치는 표시하지 않는다.
멀쩡한 표정
약간 울상
눈물 찔끔
엉엉 우는 모습
쓰러짐
전투 결과
승리: 스토리 계속 진행
패배: 전투 직전 지점으로 복귀
4.3 호감도 시스템
구조
총 10단계
단계마다 능력치 보정 증가 (전투 보조 효과)
3단계 도달: 호감도 스토리 1 해금
7단계 도달: 호감도 스토리 2 해금
10단계 도달: 호감도 스토리 3 해금 + 개인 스토리 해금
호감도 획득 방법
상점에서 호감도 아이템 구매 후 사용
황금 고블린 처치 시 호감도 아이템 드랍
호감도 스토리 특징
📌 호감도 이벤트는 이미 어느 정도 친해진 상태에서 시작한다. 처음부터 유대감이 형성된 상태의 에피소드.
4.4 황금 고블린
훈련 화면에서 랜덤 등장 — 시간당 1~5회 (완전 랜덤)
출현 후 1분간 화면 여기저기를 돌아다님

4.5 훈련도구 시스템
방치 수익(시간당 경험치 자동 적립량)을 강화하는 아이템.
접근 경로: 메인 화면 우측 상단 훈련 버튼
일반~상급: 게임 내 골드로 구매 (영구)
최상급: 현금으로 구매 (기간제)
일회성 구매, 즉시 방치 수익 증가
상점이 모든 등급을 보유하고 있음
4.6 골드 시스템
메인 화면 우측 상단 골드 버튼 탭 → 골드 획득
탭 시 골드 파티클 튀김 등 시각적 피드백 (탭 반응 필수)
골드 용도: 훈련도구 구매
4.7 탑 시스템
캐릭터별 독립 스토리 — 5명(4신수 + 가짜 황룡) 각자 다른 내러티브
각 캐릭터의 성격상 약점(싫어하는 것)과 관련된 적들이 등장
그 적들을 쓰러트리며 극복하는 성장 스토리
전투 방식: 메인 스토리와 동일 (2d6 내부 롤 + 텍스트 출력)
최대 층수 고정 → 업데이트로 확장
자기 최고 기록 갱신 방식 (랭킹 없음, P2W 컴플레인 방지)
탑 클리어 보상
공격 보정 포인트
회피 보정 포인트
방어 보정 포인트
5. UI 구조
5.1 전체 구조
하단 탭바
메인 / 스토리 / 신수 / 탑  (총 4개)
우측 상단 버튼
상점 → 골드 → 훈련  (세로로 배치, 탭바 아님)
상점 접근
메인 화면 우측 상단 상점 버튼
5.2 메인 화면 (탭바: 메인)
화면 구성
현재 선택된 신수 어린이 캐릭터가 화면 중앙에 표시
신수 터치 시 랜덤 대사 출력
대사 풀은 호감도 단계 / 성급에 따라 달라짐
왜소한 신이 화면 구석에 조용히 존재
신수 선택
버려진 신수들도 선택 가능
버려진 신수 터치 시: 현재 어떻게 살고 있는지 근황 대사 출력
현재 키우는 신수와 버려진 신수 간 시각적 구분 없음 (자연스럽게 선택)
우측 상단 버튼 (세로 배치)
상점 버튼
골드 버튼 — 탭하면 골드 획득 + 시각적 피드백
훈련 버튼 — 탭하면 훈련 화면으로 전환
5.3 훈련 화면
📌 훈련 화면은 메인 화면 우측 상단 훈련 버튼으로 접근. 탭바에 없음.
신수 SD 애니메이션: 현재 레벨에 맞는 훈련 모습 (레벨 구간마다 다른 훈련 장면)
경험치 바 표시
탭하면 경험치 획득 + SD 애니메이션 반응
황금 고블린: 랜덤으로 화면에 등장, 1분간 돌아다님, 터치로 처치
훈련도구 버튼: 방치 수익 강화 도구 관리
5.4 스토리 화면 (탭바: 스토리)
비주얼노벨 스타일 텍스트 진행
적 등장 시 탭 버튼 출현
전투 중: 신수 표정 이미지 + 전투 텍스트 + 탭 버튼
탭 1회 = 내부 다이스 롤 → 결과에 맞는 텍스트 자동 출력 (다이스 수치 비표시)
적 상태: 텍스트로만 표현 (HP 수치 비표시)
캐릭터 HP: 표정 변화로 표현 (수치 비표시)
승리 → 스토리 계속 / 패배 → 전투 직전으로 복귀
5.5 신수 화면 (탭바: 신수)
5장의 신수 카드 목록
카드 탭 → 신수 상세 페이지 진입
신수 상세 페이지
신수 카드 일러스트 (고퀄리티 어린이 캐릭터)
기본 설명 — 처음부터 공개
TMI — 성급 오를 때마다 하나씩 해금
호감도 바
호감도 아이템 선물 버튼
호감도 10단계: 개인 스토리 해금
5.6 탑 화면 (탭바: 탑)
캐릭터 선택
현재 층수 및 내 최고 기록 표시
전투 방식: 스토리 화면과 동일 구조
층 클리어 시 능력치 보정 보상

6. 수익화 모델
6.1 스킨
캐릭터당 5개 내외
상점에서 직접 구매 (유료)
황금 고블린 스킨 가챠권으로도 획득 가능 (랜덤 or 선택권 두 종류)
테마 예시: 계절 버전, 일상 버전, 각성 후 일상 버전 등
6.2 호감도 아이템
상점에서 골드 또는 현금으로 구매
황금 고블린 드랍으로도 획득
6.3 훈련도구
일반~상급: 게임 내 골드로 구매 (영구)
최상급: 현금 결제, 기간제
6.4 스킨 가챠권
황금 고블린 희귀 드랍
종류 1: 랜덤 스킨
종류 2: 선택권 (원하는 스킨 교환)
7. 아트 방향
📌 배경 이미지, 적 이미지 없음. 아트 리소스를 전부 캐릭터 카드 일러스트에 집중한다.
모든 신수: 어린이 모습 2D 일러스트
각성 전후: 외형 동일, 이펙트만 차이
신수 카드 일러스트 = 게임의 첫인상이자 핵심 자산. 퀄리티 최우선
SD 훈련 애니메이션: 레벨 구간마다 다른 귀여운 훈련 장면
표정 변화: 멀쩡 / 약간 울상 / 눈물 찔끔 / 엉엉 / 쓰러짐 (HP 표현용)
